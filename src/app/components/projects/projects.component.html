GroupThink

Made with: Java, Spring Boot, PostgreSQL, Angular, Maven, Git

GroupThink is a project management application that lets you organize teams and projects. Adminsitrators can easily create dynamic teams, assign them new projects, or edit existing ones. New users can be added to the company, and GroupThink supports enrollement in multiple companies. All changes are persisted in a PostgreSQL database by the use of a Spring Boot RESTful API.

Twitter RESTful API

Made with: Java, Spring Boot, Maven, PostgreSQL, Git, Postman

A Twitter backend API clone with the capabilities to handle requests such as creating new users, creating tweets, liking tweets, and creating/applying hashtags. All incoming http requests are recieved as DTOs by our controller layer, processed by our service layer, and saved as entities into our PostgreSQL database via our repository layer.

Apartments Database

Made with: OracleSQL, SQLPlus, Git

This is a relational database that represents a series of apartment buildings and their apartments, built from the ground up starting with only the client's written requirements. Apartment buildings, apartments, and apartment features are created, read, updated, and deleted with the use of various SQL queries. More complex features such as joins, views, and nested queries are used to satisfy the client's requests. Documentation that contains a Data Dictionary and ER Diagrams can be found in this project's github repository.

Spotify Quiz Game Application

Made with: Angular, Typescript, npm, Git

Test your knowledge of the greatest hits in music history with this time attack style quiz. You are given a short sample of a popular song provided by the Spotify API, and you are given 4 random track titles for you to guess what song is playing. Guess as many songs as you can in 60 seconds, but be careful! If you run out of lives, it's game over. Your score is saved in a leaderboard so that you may cement yourself as the #1 pop music nerd.
